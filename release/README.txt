WOAPro v1.1 - Release Package
==============================

This package contains the standalone executable version of WOAPro (Work Order Analysis Professional).

CONTENTS:
- WOAPro.exe (299MB) - Main application executable
- app_config_template.json - Configuration template
- failure_mode_dictionary_.xlsx - Sample failure mode dictionary

INSTALLATION:
1. Extract all files to a folder of your choice
2. Copy app_config_template.json to app_config.json
3. Update the paths in app_config.json to point to your data files
4. Run WOAPro.exe

SYSTEM REQUIREMENTS:
- Windows 10/11 (64-bit)
- 4GB RAM minimum, 8GB recommended
- 500MB free disk space

USAGE:
1. Double-click WOAPro.exe to start the application
2. Load your work order data file (Excel or CSV format)
3. Configure column mappings if needed
4. Use the various analysis tabs to perform reliability analysis
